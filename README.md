# HTB VPN Logging PS1

Prompt personalizado de Bash diseñado para entornos de pentesting como Hack The Box.

## Funcionalidades

- Detección automática de VPN HTB
- Visualización de servidor e IP VPN
- Timestamp dinámico en cada línea
- Prompt en dos líneas con colores
- Script para logging automático de sesiones

## Uso

Cargar el prompt:

```bash
source custom-ps1.sh
